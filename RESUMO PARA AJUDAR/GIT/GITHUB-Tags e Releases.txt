Tags e Releases

Salvar uma versão do projeto no github

Comando:
git tag -a v0.1.0 

Colocando uma mensagem na versão

Comando:
git tag -a v0.1.0 -m "mensagem-para-falar-do-que-significa-a-versão"

Após, roda o comando abaixo, vai mostrar todos as versões disponíveis. 

Comando:
git tag

Adicionar a versão no github

Comando:
git push origin v0.1.0
